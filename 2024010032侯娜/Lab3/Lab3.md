## 第 20 题（问答/证明题，3 分）

**PRG 优势（Advantage）的定义与计算**

设 $G: K \to \{0,1\}^n$ 是一个 PRG，$A$ 是 $\{0,1\}^n$ 上的统计测试。

**(a)** 请写出 PRG 优势 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 的定义式。

**答案：**

```

设  G: K \to \{0,1\}^n  是一个伪随机生成器（PRG）， A  是作用在  \{0,1\}^n  上的一个统计测试（即区分器）。则 PRG 优势定义为：

\text{Adv}_{\text{PRG}}[A, G] = \left\Pr_{k \xleftarrow{\} K}[A(G(k)) = 1] - \Pr_{r \xleftarrow{\} \{0,1\}^n}[A(r) = 1] \right

或等价写作：

\text{Adv}_{\text{PRG}}[A, G] = \left\Pr[A(G(k))=1] - \Pr[A(r)=1] \right

其中：
•  k \xleftarrow{\} K ：密钥  k  均匀随机选取自密钥空间  K $；

•  r \xleftarrow{\} \{0,1\}^n ：真随机串  r  均匀随机选取自  \{0,1\}^n $；

•  A(\cdot)  输出为 0 或 1，表示区分器“认为输入是伪随机”或“认为是真随机”。

```

**(b)** 解释 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 接近 1 和接近 0 分别意味着什么。

**答案：**

```
• 当  \text{Adv}_{\text{PRG}}[A, G]  接近 1：  

  表示区分器  A  能够高度准确地区分  G(k) （伪随机输出）与真随机串  r 。这意味着 PRG  G  不安全，其输出可被有效识别为非随机，违背了伪随机性的基本要求。

• 当  \text{Adv}_{\text{PRG}}[A, G]  接近 0：  

  表示区分器  A  几乎无法区分伪随机输出与真随机串，其成功概率与随机猜测无异。这意味着 PRG  G  安全，在计算上不可区分，满足伪随机生成器的设计目标。
```

**(c)** 若 $A(x) = 0$（即 $A$ 对任何输入都输出 0），求 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 的值。

**答案：**

```
已知：对任意输入  x ，都有  A(x) = 0 ，即：

•  Pr[A(G(k)) = 1] = 0 

•  Pr[A(r) = 1] = 0 

代入定义式：

\text{Adv}_{\text{PRG}}[A, G] = \left0 - 0 \right
 = 0

结论：  
当区分器  A  恒输出0时，其优势为 0。  
这表示该区分器完全无区分能力——虽然它“安全”，但只是因为“什么都不做”，并非 PRG 本身安全，而是测试无效。
```

## 第 21 题（问答/证明题，4 分）

**PRG 优势的具体计算**

假设 $G: K \to \{0,1\}^n$ 满足：对于 $K$ 中 $\frac{2}{3}$ 的密钥 $k$，有 $\mathrm{MSB}(G(k)) = 1$（即输出的最高有效位为 1）。

定义统计测试 $A(x)$ 如下：
$$A(x) = \begin{cases} 1 & \text{若 } \mathrm{MSB}(x) = 1 \\ 0 & \text{否则} \end{cases}$$

**(a)** 计算 $\Pr[A(G(k)) = 1]$，其中 $k \xleftarrow{R} K$。

**答案：**

```  
已知条件：对于 K 中 2/3 的密钥 k，有 MSB(G(k)) = 1  
由于 A(x) = 1 当且仅当 MSB(x) = 1，因此：
Pr[A(G(k)) = 1] = Pr[MSB(G(k)) = 1] = 2/3
```

**(b)** 计算 $\Pr[A(r) = 1]$，其中 $r \xleftarrow{R} \{0,1\}^n$。

**答案：**

```
对于真随机串 r \in \{0,1\}^n，每一位都是独立均匀随机的，因此最高有效位为 0 或 1 的概率均为 1/2：
Pr[A(r) = 1] = Pr[MSB(r) = 1] = 1/2
```

**(c)** 计算 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$。

**答案：**

```
代入 (a)(b) 的结果：
text{Adv}_{PRG}[A, G] = \left\frac{2}{3} - \frac{1}{2} \right = \left \frac{4}{6} - \frac{3}{6} \right
 = 1/6
```

**(d)** 该 PRG 是否安全？为什么？

**答案：**

```
不安全
理由：text{Adv}_{PRG}[A, G] = 1/6 是一个常数优势，不是可忽略的函数。在密码学中，安全的 PRG 要求所有多项式时间的区分器优势都是可忽略的（即随安全参数增加而快速趋近于 0）。常数优势表明该 PRG 存在可被有效检测的统计偏差，不满足伪随机生成器的安全性要求。
```

## 第 22 题（证明题，5 分）

**证明 OTP 具有完美保密性**

**引理**：一次性密码本（OTP）具有完美保密性（Perfect Secrecy）。

**(a)** 写出完美保密性的定义：对于任意明文 $m$ 和密文 $c$，$\Pr_k[E(k,m) = c]$ 应满足什么条件？

**答案：**

```
对于任意固定的明文  m  和密文  c ，概率 \Pr_k[E(k,m) = c] 应该与  m  无关，即对所有  m_1, m_2 \in M ，有

Pr_k[E(k, m_1) = c] = Pr_k[E(k, m_2) = c]

```

**(b)** 对于 OTP，证明对任意 $m \in \mathcal{M}$ 和 $c \in \mathcal{C}$，恰好存在多少个密钥 $k$ 使得 $E(k, m) = c$？

**答案：**

```
在一次性密码本（OTP）中：

• 明文空间  M = \{0, 1\}^n 

• 密钥空间  K = \{0, 1\}^n 

• 密文空间  C = \{0, 1\}^n 

• 加密函数： E(k, m) = k \oplus m （按位异或）

给定任意明文  m \in M  和任意密文  c \in C ，我们想找出满足  E(k, m) = c  的密钥  k  的数量。

由加密公式：

k \oplus m = c \Rightarrow k = c \oplus m

因为异或运算可逆，且在  \{0, 1\}^n  中，对任意固定的  m  和  c ，有且仅有唯一一个  k = c \oplus m  满足该等式。

因此，对任意  m \in M  和  c \in C ，恰好存在 1 个密钥  k  使得  E(k, m) = c 。
```

**(c)** 利用 (a) 和 (b) 的结论，完成 OTP 具有完美保密性的证明。

**答案：**

```
由 (b) 的结论：对任意  m \in M ,  c \in C ，恰好存在唯一一个  k \in K  使得  E(k, m) = c 。

又因为密钥  k  是从密钥空间  K  中均匀随机选取的（这是OTP的基本假设），且 $ K
 = 2^n $，所以：

Pr_k[E(k, m) = c] = \frac{\text{满足 } E(k, m) = c \text{ 的密钥个数}}{K
} =1/2^n

这个概率只依赖于密钥空间大小，而与具体的明文  m  无关！

因此，对任意两个明文  m_1, m_2 \in M ，都有：

Pr_k[E(k, m_1) = c] = Pr_k[E(k, m_2) = c] = 1/2^n

这完全符合完美保密性的定义。

综上，一次性密码本（OTP）满足完美保密性。
```

## 第 23 题（证明题，5 分）

**语义安全性与明文信息泄露**

假设存在一个高效算法 $A$，它总是能从密文 $c$ 中推断出明文 $m$ 的最低有效位 $\mathrm{LSB}(m)$。

**(a)** 请证明：加密方案 $\mathbb{E} = (E, D)$ 不是语义安全的。

**答案：**

```
假设存在高效算法 A，对任意密文 c = E(pk, m)，总有 A(c) = LSB(m)。

根据语义安全的定义（IND-CPA 安全）：  
对于任意多项式时间敌手 B，其在语义安全游戏中猜中挑战比特 b 的优势满足：  
AdvSS[B, E] = Pr[b' = b] − 1/2
 ≤ negl(λ)

但若存在算法 A 总能正确输出 LSB(m)，则我们可以构造一个敌手 B，使其在 IND-CPA 游戏中以 1 的概率猜中 b（见 (b) 部分），从而 AdvSS[B, E] = 1/2，不可忽略，与语义安全矛盾。

因此，加密方案 E = (E, D) 不是语义安全的。  
```

**(b)** 构造一个攻击者 $B$，使用 $A$ 作为子程序来赢得语义安全游戏。具体地：
   - 攻击者 $B$ 应选择哪两条消息 $m_0, m_1$ 发送给挑战者？
   - 收到密文 $c$ 后，$B$ 如何利用 $A$ 来猜测 $b$？

**答案：**

```
• 攻击者 B 应选择：m₀ = 0，m₁ = 1（或任选 LSB 不同的两消息）  

• 收到密文 c 后，B 调用子程序 A(c) 得到 LSB(m_b)，并令猜测值 b' = A(c)。由于 LSB(m₀)=0，LSB(m₁)=1，故 b' = b 恒成立。
```

**(c)** 计算 $\mathrm{Adv}_{\mathrm{SS}}[B, \mathbb{E}]$ 的值。

**答案：**

```
由定义得：
AdvSS[B, E] = Pr[b' = b] − 1/2

由 (b) 可知：  
Pr[b' = b] = 1 （因为 B 利用 A 总能准确恢复 LSB(m_b)，而 LSB(m_b) = b）

因此：  
AdvSS[B, E] = 1 − 1/2
 = 1/2
```

## 第 24 题（证明题，5 分）

**证明 OTP 是语义安全的**

考虑语义安全游戏的两个实验：
- **EXP(0)**：挑战者加密 $m_0$，即 $c \leftarrow k \oplus m_0$
- **EXP(1)**：挑战者加密 $m_1$，即 $c \leftarrow k \oplus m_1$

其中 $k \xleftarrow{R} K$，$|m_0| = |m_1|$。

**(a)** 证明在 EXP(0) 中，密文 $c = k \oplus m_0$ 在 $\{0,1\}^n$ 上是均匀分布的。

**答案：**

```

证明：

在实验 EXP(0) 中， c = k \oplus m_0 ，其中：
•  k \xleftarrow{\} \{0,1\}^n $ 是均匀随机选取的密钥

•  m_0  是固定明文（挑战者选择）

设 y \in \{0,1\}^n 为任意固定长度为 n 的比特串。计算 c = y 的概率：

Pr[c = y] = Pr[k \oplus m_0 = y] = Pr[k = y \oplus m_0]

由于 k 是在\{0,1\}^n上均匀随机的，且 y \oplus m_0 是一个确定的比特串，所以：

Pr[k = y \oplus m_0] =1/2^n

这个概率对所有的 y \in \{0,1\}^n 都相同，且与 m_0 无关。

结论：
在 EXP(0) 中，密文 c 是\{0,1\}^n上的均匀分布。
```

**(b)** 证明在 EXP(1) 中，密文 $c = k \oplus m_1$ 在 $\{0,1\}^n$ 上也是均匀分布的。

**答案：**

```
证明：

在实验 EXP(1) 中， c = k \oplus m_1 ，其中：
•  k \xleftarrow{\} \{0,1\}^n $ 是均匀随机选取的密钥

•  m_1  是固定明文（挑战者选择）

设 y \in \{0,1\}^n 为任意固定长度为 n 的比特串。计算 c = y 的概率：

Pr[c = y] = Pr[k \oplus m_1 = y] = Pr[k = y \oplus m_1]

由于 k 是在\{0,1\}^n上均匀随机的，且 y \oplus m_1 是一个确定的比特串，所以：

Pr[k = y \oplus m_1] = 1/2^n

这个概率对所有的 y \in \{0,1\}^n 都相同，且与 m_1 无关。

结论：
在 EXP(1) 中，密文 c 也是\{0,1\}^n上的均匀分布。
```

**(c)** 由 (a) 和 (b) 说明 EXP(0) 和 EXP(1) 中密文的分布是相同的（identical distributions）。

**答案：**

```
由(a)可知：在 EXP(0) 中，对任意 y \in \{0,1\}^n ：

Pr[c = y] = 1/2^n

由(b)可知：在 EXP(1) 中，对任意 y \in \{0,1\}^n ：

Pr[c = y] = 1/2^n

比较：
两个概率表达式完全相同，说明：
1. 两个实验中密文的概率分布函数相同
2. 对任意可能的密文输出 y ，其在两个实验中被生成的概率相等

结论：
 EXP(0) 和 EXP(1) 中密文的分布是相同的（identical distributions）。
 从概率论角度，这两个分布是不可区分的，因为它们的概率质量函数完全一致。
```

**(d)** 利用 (c) 的结论，证明对于任意攻击者 $A$：
$$\mathrm{Adv}_{\mathrm{SS}}[A, \mathrm{OTP}] = \left| \Pr[A(k \oplus m_0) = 1] - \Pr[A(k \oplus m_1) = 1] \right| = 0$$

**答案：**

```
证明：

1. 设随机变量 C_0 = k \oplus m_0 （ EXP(0) 中的密文）
2. 设随机变量 C_1 = k \oplus m_1 （ EXP(1) 中的密文）

由(c)的结论可知， C_0 和 C_1 的分布相同，即：

C_0 \stackrel{d}{=} C_1

其中 \stackrel{d}{=} 表示分布相等。

3. 对于任意攻击者 A ，其输出仅依赖于输入的密文 c 。因此：
   •  Pr[A(k \oplus m_0) = 1] = Pr[A(C_0) = 1] 

   •  Pr[A(k \oplus m_1) = 1] = Pr[A(C_1) = 1] 

4. 由于 C_0 和 C_1 分布相同，对任意函数 A （包括任何高效攻击者），有：
   
   Pr[A(C_0) = 1] = Pr[A(C_1) = 1]
   

5. 代入优势定义：
   
   \begin{aligned}
   \text{Adv}_{SS}[A, OTP] &= \left\Pr[A(k \oplus m_0) = 1] - \Pr[A(k \oplus m_1) = 1] \right
 \\
   &= \left\Pr[A(C_0) = 1] - \Pr[A(C_1) = 1] \right
 \\
   &= X - X
 \quad \text{（其中 X = \Pr[A(C_0) = 1] = \Pr[A(C_1) = 1] ）} \\
   &= 0
   \end{aligned}
   

结论：
对于任意攻击者 A （无论多么强大，包括计算无界的攻击者），其语义安全优势都为：
\boxed{\text{Adv}_{SS}[A, OTP] = 0}
```
