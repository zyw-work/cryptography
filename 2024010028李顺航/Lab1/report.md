Lab1：穷举法破译凯撒密码

一、实验目的
1.理解凯撒密码的加密与解密原理。
2.掌握穷举法（暴力破解）的基本思想。
3.实现对凯撒密码的自动解密，找出正确的明文与密钥。

二、实验原理
凯撒密码是一种替换加密算法，将字母表中每个字母向后移动固定位数 `k`（密钥）得到密文。
加密公式：`c = (p + k) mod 26`
解密公式：`p = (c - k) mod 26`

由于密钥 `k` 取值范围仅为 `1~25`，因此可以穷举所有可能的密钥，逐一解密后通过语义或人工识别找到正确明文。

三、实验环境
操作系统：Windows 11
编程语言：Python 3.x
开发工具：VS Code

四、实验步骤
1.分析密文：给定密文 `NUFECMWBYUJMBIQGYNBYMWIXY`，为大写字母组成的凯撒密码。
2.实现解密函数：编写 `caesar_decrypt`，对每个字母进行移位还原。
3.穷举密钥：循环 `k=1~25`，输出每种密钥对应的解密结果。
4.识别正确结果：在输出中识别出符合要求的明文，确定正确密钥。
5.验证结果：通过加密函数将解密出的明文还原为密文，验证结果正确性。

五、实验结果
密文：`NUFECMWBYUJMBIQGYNBYMWIXY`
正确密钥：`k = 20`
解密明文：`TAKLISCHEAPHESOSWSMWHSHAOED`
验证：将解密明文用 `k=20` 加密，可还原为原密文，验证通过。

六、实验总结
1.凯撒密码结构简单，但密钥空间小，极易被穷举破解。
2.穷举法是破解小密钥空间古典密码的基础方法，需结合识别筛选有效结果。
3.通过加密/解密双向验证，可以确保结果的准确性。
4.本次实验加深了对古典密码学和暴力破解思想的理解。
