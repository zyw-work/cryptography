## 第 22 题（证明题，5 分）

**证明 OTP 具有完美保密性**

**引理**：一次性密码本（OTP）具有完美保密性（Perfect Secrecy）。

**(a)** 写出完美保密性的定义：对于任意明文 $m$ 和密文 $c$，$\Pr_k[E(k,m) = c]$ 应满足什么条件？

**答案：**

```
对于任意明文 m₀, m₁ ∈ 𝒫 和任意密文 c ∈ 𝒞，满足：
Pr[E(k, m₀) = c] = Pr[E(k, m₁) = c]
其中 k 是从密钥空间 𝒦 中随机选取的密钥。
等价表述：对于任意 m, c ∈ 𝒫 × 𝒞，Pr_k[E(k, m) = c] = 1/|𝒦|。
```

**(b)** 对于 OTP，证明对任意 $m \in \mathcal{M}$ 和 $c \in \mathcal{C}$，恰好存在多少个密钥 $k$ 使得 $E(k, m) = c$？

**答案：**

`恰好存在 1 个密钥 k = m ⊕ c 使得 E(k, m) = c。``

```

**(c)** 利用 (a) 和 (b) 的结论，完成 OTP 具有完美保密性的证明。

**答案：**

```
证明过程如下：
1. 设 OTP 的密钥空间 𝒦 = 消息空间 𝒫 = 密文空间 𝒞 = {0,1}^n，密钥 k 均匀随机选取。
2. 对任意明文 m ∈ 𝒫 和密文 c ∈ 𝒞，由 (b) 知，存在唯一密钥 k = m ⊕ c 使得 E(k, m) = c。
3. 密钥 k 均匀分布，因此 Pr_k[E(k, m) = c] = 1/|𝒦| = 1/2^n。
4. 该概率与明文 m 无关，满足完美保密性的定义。
因此，一次性密码本（OTP）具有完美保密性。
```

## 第 24 题（证明题，5 分）

**证明 OTP 是语义安全的**

考虑语义安全游戏的两个实验：
- **EXP(0)**：挑战者加密 $m_0$，即 $c \leftarrow k \oplus m_0$
- **EXP(1)**：挑战者加密 $m_1$，即 $c \leftarrow k \oplus m_1$

其中 $k \xleftarrow{R} K$，$|m_0| = |m_1|$。

**(a)** 证明在 EXP(0) 中，密文 $c = k \oplus m_0$ 在 $\{0,1\}^n$ 上是均匀分布的。

**答案：**

```
证明：设 c = k ⊕ m₀，其中 k ←_R {0,1}^n，m₀ 为固定消息。
对任意固定的密文 c₀ ∈ {0,1}^n，计算 Pr[c = c₀]：
Pr[k ⊕ m₀ = c₀] = Pr[k = c₀ ⊕ m₀] = 1/2^n
（因为 k 是均匀随机的，对任意固定值，概率为 1/2^n）。
由于该概率与 c₀ 无关，因此 c 在 {0,1}^n 上均匀分布。
```

**(b)** 证明在 EXP(1) 中，密文 $c = k \oplus m_1$ 在 $\{0,1\}^n$ 上也是均匀分布的。

**答案：**

```
证明：设 c = k ⊕ m₁，其中 k ←_R {0,1}^n，m₁ 为固定消息。
对任意固定的密文 c₁ ∈ {0,1}^n，计算 Pr[c = c₁]：
Pr[k ⊕ m₁ = c₁] = Pr[k = c₁ ⊕ m₁] = 1/2^n
（同理，k 均匀随机，任意固定值的概率为 1/2^n）。
该概率与 c₁ 无关，因此 c 在 {0,1}^n 上均匀分布。
```

**(c)** 由 (a) 和 (b) 说明 EXP(0) 和 EXP(1) 中密文的分布是相同的（identical distributions）。

**答案：**

```
由 (a) 和 (b) 可知，EXP(0) 中的密文 c 服从 {0,1}^n 上的均匀分布，EXP(1) 中的密文 c 也服从 {0,1}^n 上的均匀分布。
两个实验的密文分布均为均匀分布，因此它们的分布是相同的（identical distributions）。
```

**(d)** 利用 (c) 的结论，证明对于任意攻击者 $A$：
$$\mathrm{Adv}_{\mathrm{SS}}[A, \mathrm{OTP}] = \left| \Pr[A(k \oplus m_0) = 1] - \Pr[A(k \oplus m_1) = 1] \right| = 0$$

**答案：**

```
证明：语义安全优势定义为：
Adv_SS[A, OTP] = | Pr[A(EXP(0)) = 1] - Pr[A(EXP(1)) = 1] |
其中 EXP(0) 输出 c = k ⊕ m₀，EXP(1) 输出 c = k ⊕ m₁，k ←_R {0,1}^n。
由 (c) 知，EXP(0) 和 EXP(1) 中的密文 c 均服从均匀分布，因此对于任意攻击者 A，Pr[A(c)=1] 的值在两个实验中相等。
即：
Pr[A(k ⊕ m₀) = 1] = Pr[A(k ⊕ m₁) = 1]
因此，Adv_SS[A, OTP] = | Pr[A(k ⊕ m₀) = 1] - Pr[A(k ⊕ m₁) = 1] | = 0。
由于该优势对任意攻击者 A 都成立，因此 OTP 是语义安全的。
