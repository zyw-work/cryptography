# 流密码多次密钥攻击实验报告

## 一、分析方法
1. 所有密文使用同一密钥流 K 加密：Cᵢ = Mᵢ ⊕ K
2. 任意两个密文异或：Cᵢ⊕Cⱼ = Mᵢ⊕Mⱼ，消除密钥流
3. 利用空格(0x20)与字母异或会翻转大小写的特性，逐位置投票猜测密钥流字节
4. 用投票得到的密钥流解密目标密文

## 二、密钥流推断过程
- 遍历每个字节位置
- 对所有密文对异或，统计符合“大小写字母”特征的候选密钥
- 取票数最高的字节作为该位置密钥
- 最终得到完整密钥流

## 三、解密结果
目标密文解密后的明文为：

The secret message is: When using a stream cipher, never use the same key more than once.
